# Linked list
Lab 2, Methodologies and technologies of software development
## Getting started
1. Clone repo
```
git clone https://github.com/chyrkovmaksym/Chyrkov_MTRPZ_lab2.git
```
2. Run 
```
npm i
```
3. To run project: 
```
npm start
```
4. To run tests:
```
npm run test
```
